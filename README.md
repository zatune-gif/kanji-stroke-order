# 漢字書き順 — 書き順なぞりゲーム

書き順をなぞって小学漢字を覚えるスマートフォン向けゲームです。  
**インストール不要。URLを開くだけでそのまま遊べます。**  
ホーム画面に追加するとアドレスバーなしのアプリとして使えます（任意）。

**▶ プレイする:** https://zatune-gif.github.io/kanji-stroke-order/

---

## 概要

| 項目 | 内容 |
|---|---|
| 対象 | 小学1〜6年生（学習漢字 1,006字） |
| 動作環境 | スマートフォン／タブレット（iOS Safari・Android Chrome 推奨） |
| インストール | 不要（ブラウザで即プレイ） |
| オフライン | ホーム画面に追加後はオフラインでもプレイ可能 |
| データ保存 | クリア記録はブラウザの localStorage に自動保存 |
| 料金 | 無料 |

---

## 遊び方

### ゲームモード

1. 学年（1〜6年）と問題数（5・10・15・20問）を選ぶ
2. 漢字の書き順アニメーションを見る（「書き順を見てね！」と表示される）
3. ガイド枠に表示された方向に指でなぞる（1画ずつ）
4. 全問終了後に総合結果が発表される

### 自由練習モード

- 学年・漢字を自由に選んで1文字ずつ練習できる
- 練習でクリアした漢字もゲームの進捗に反映される

### 判定

- 指でなぞった方向が正解と一致すれば **◯**
- 方向が違うと **✕**（ミスとしてカウント）
- ◯✕どちらの場合も、正しい画が薄く表示されるので書き順を確認できる

### 評価（全問終了後）

| 評価 | 条件 |
|---|---|
| パーフェクト！ | 全問ミスなし |
| もうちょっと！ | 全問ミスあり、「がんばろう」なし |
| がんばろう！ | 3ミス以上の問題が1問でもある |

### クリア記録

- クリアした漢字は次回ゲームで出題されなくなる
- 学年内の全漢字をクリアすると記録がリセットされ最初から出題される
- 設定画面でいつでもリセット可能（学年ごと・全学年）

## ホーム画面への追加方法（任意・アプリとして使う場合）

### iPhone / iPad（Safari）

1. Safari でゲームの URL を開く
2. 画面下の共有ボタン（↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

ホーム画面のアイコンからタップすると、アドレスバーなしのフルスクリーンで起動します。

> ※ iOS では Safari のみ対応。Chrome・Firefox では追加できません。

### Android（Chrome）

1. Chrome でゲームの URL を開く
2. 右上メニュー（⋮）→「ホーム画面に追加」または「アプリをインストール」
3. 「追加」をタップ

## 仕様

### 音声

- BGM：タイトル・メニュー・ゲーム・自由練習でそれぞれ異なる曲が流れる
- 効果音：正解音・ミス音（Web Audio API 生成）
- 結果ジングル：総合評価に応じた3種類のジングルが鳴る
- ミュートボタンで全音声をOFF/ON切替可能（設定は保存される）

### 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- Canvas API（書き順描画・なぞり入力）
- Web Audio API（効果音・ジングル）
- HTML Audio（BGM）
- Service Worker（オフライン対応・キャッシュ）
- localStorage（クリア記録・ミュート設定の保存）

### 漢字データ

文部科学省が定める**小学校学習指導要領の学習漢字 1,006字**を収録しています。

各漢字の筆順・画の座標データは [KanjiVG](https://kanjivg.tagaini.net/)（Ulrich Apel 氏作成、CC BY-SA 3.0）を使用しています。KanjiVG は SVG 形式で各漢字の筆順パスを公開しており、本ゲームではそのパスデータをもとに書き順アニメーションとなぞり判定を実現しています。
